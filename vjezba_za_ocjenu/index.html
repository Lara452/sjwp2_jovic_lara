HTML, CSS, Canvas i SVG

Canvas je HTML5 element koji, pomoću JavaScript-a, omogućava dinamičko iscrtavanje različitih oblika, dodavanje slika i kreiranje animacija na web stranici.

Na platnu (canvas-u) to ne radimo korištenjem novog jezika (kao što je to slučaj kod SVG) ili korištenjem Flasha. Najčešće uporabe platna su:

  • crtanje grafova
  • animacije i igre
  • kompozicija slike

SVG (Scalable Vector Graphics) grafika ne gubi kvalitetu prilikom zumiranja ili promjene veličine.

Svi crteži mogu biti interaktivni i dinamički. Svaki element i atribut SVG datoteke može biti animiran, definirano i pokrenuto interaktivno ili pomoću skripti.

SVG integrira druge W3C standarde kao što su DOM, XSL itd.

Usporedba SVG-a i Canvasa

SVG HTML Canvas
SVG ima bolju skalabilnost i zadržava kvalitetu slike pri svakoj rezoluciji. Canvas ima lošiju skalabilnost i nije prikladan za vrlo visoke rezolucije.
SVG daje bolje performanse pri manjem broju objekata i većoj površini. Canvas daje bolje performanse pri većem broju objekata i manjim površinama.
SVG se može uređivati pomoću CSS-a i skripti. Canvas se može uređivati isključivo pomoću skripte.
SVG je baziran na vektorima i sastavljen od oblika. Canvas je baziran na pikselima.
